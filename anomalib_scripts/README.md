## Installation Guide

**prerequisites:** 
- make pytorch enabled to use GPU 
- run [gpu_vs_cpu_test.py](test_scripts/gpu_vs_cpu_test.py) to verify if pytorch is enabled to use GPU 

**anomalib installation:**

    pip install anomalib
    anomalib install -v
